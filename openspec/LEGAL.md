# LEGAL.md — 外观设计专利侵权判定法律依据

## 法律框架

### 中国专利法（CNIPA 标准）

**来源**：国家知识产权局《外观设计专利侵权判定实务》、最高人民法院司法解释

| 维度 | 标准 | 条文依据 |
|------|------|----------|
| **产品种类** | 被控产品与专利产品用途相同或相近 | 种类不同则直接不侵权 |
| **整体视觉效果** | 整体观察、综合判断，不以局部细节为准 | 司法解释第11条 |
| **区别设计特征** | 区别于现有设计的创新点被复制 = 强侵权信号 | 司法解释第11条 |
| **设计空间** | 该品类设计自由度越大，细微差别越易被忽略 | CNIPA 指南 |
| **易观察部位** | 正常使用时直接可见部位影响更大 | CNIPA 指南 |
| **功能性排除** | 纯功能决定的设计特征不纳入比对 | 司法解释 |

### 美国标准（Ordinary Observer Test）

**来源**：Egyptian Goddess, Inc. v. Swisa, Inc. (Fed. Cir. 2008)

以普通消费者视角，判断被控产品是否会让消费者误认为与专利设计来自同一来源。整体视觉效果无实质性差异 = 侵权。

### 判定方法

1. **直接观察** — 肉眼比对，不做仪器分析
2. **单独对比** — 被控产品 vs 单个专利，不做组合比较
3. **整体视觉效果** — 综合形状、图案、色彩
4. **排除功能性设计** — 纯功能决定的特征不计入

## 操作定义（法律 → 工程转换）

### 正样本（侵权）生成规则

**法律要件**：同类产品 + 整体视觉效果实质性相似 + 区别设计特征被保留 → 侵权

**生成规则**：
- 同品类产品（从同一搜索关键词获取）
- 保留核心设计特征：主体轮廓、比例、结构布局不变
- 仅改变非核心细节：颜色/材质/纹理/小装饰元素
- 变换后与原图相似度应保持在 **50-75% 之间**

### 负样本（不侵权）生成规则

**法律要件**：任一下列条件成立 → 不侵权
1. 不同类产品
2. 同类但整体视觉效果显著不同
3. 同类且整体相似，但核心区别设计特征不同

**生成规则**：
- 同品类不同专利号（相同搜索关键词）
- 主体外形、比例、结构必须与原注册图不同
- 与任意注册图相似度 **≤ 75%**（pipeline 内置门禁验证）
- 不同类别产品（Phase 2 扩展）

### 相似度阈值依据

基于 CNIPA"整体视觉效果"标准 + 美国"Ordinary Observer"标准：
- **>75% 相似**：一般消费者可能混淆 → 应判侵权（正样本区域）
- **≤75% 相似**：一般消费者能区分差异 → 应判不侵权（负样本区域）

## 典型案例参考

### 构成侵权的案例
- **Apple v. Samsung**：圆角矩形 + Home 键布局 → 整体视觉效果相同 → 侵权
- **戴森 v. 某吸尘器**：整体造型 + 色彩搭配高度相似 → 侵权
- **萌猫保温杯案**：区别设计特征（猫耳朵+胡须+尾巴）被复制 → 侵权

### 不构成侵权的案例
- **不同类产品**：水杯外观 vs 台灯外观 → 不侵权
- **功能决定设计**：螺丝螺纹、散热孔布局 → 不纳入比对
- **核心区别特征不同**：同为保温杯，但杯盖核心设计特征完全不同 → 不侵权
