# PRODUCT.md — 侵权检测训练图库生成器

## 产品愿景

构建一个自动化训练图库生成系统，为侵权检测 AI 模型提供高质量的正负样本数据集。覆盖六种侵权类型，通过程序化变换与 AI 生成相结合的方式，产出可用于训练深度学习模型的标注图片数据。

## 目标用户

- 侵权检测 AI 模型训练者
- 知识产权保护平台开发者
- 电商平台内容审核系统

## 侵权类型覆盖（6 类）

| 编号 | 侵权类型 | 说明 |
|------|----------|------|
| T1 | 盗图 | 直接复制或轻微修改他人图片 |
| T2 | 外观设计专利 | 产品外观与已注册专利设计实质性相似 |
| T3 | 美术作品著作权 | 绘画/插画/设计稿被复制或改编 |
| T4 | 文字商标权 | 未经授权使用已注册商标文字 |
| T5 | 图案商标权 | 未经授权使用已注册商标图形 |
| T6 | 肖像权 | 未经授权使用他人肖像 |

## 数据结构

```
datasets/
├── registry/                    # 注册图库（比对基准）
│   └── {侵权类型}/
│       └── {图片文件}
│
└── training/                    # 训练样本
    └── {侵权类型}/
        ├── positive/            # 侵权样本
        │   └── {图片文件}
        └── negative/            # 不侵权样本
            └── {图片文件}
```

## 成功标准

### Phase 1 — 当前目标
- 仅实现外观设计专利（T2）类型
- 注册图库：从公开专利数据库爬取 ≥5 张
- 训练样本：正样本 20 张（程序化变换生成）+ 负样本 20 张
- 能在本地跑通完整流程

### Phase 2 — 规模化
- 扩展全部 6 种侵权类型
- 每类 1000 张训练样本
- 接入 ComfyUI 做 img2img 增强真实感
- 输出 PyTorch Dataset 可直接训练
